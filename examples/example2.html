# Overhead Example ## converting from Markdown to HTML on the fly
# Check the source The source of this file is actually Markdown (with some HTML decoration).
# Magic! - Markdown is converted to HTML on the fly by JavaScript. - I use [CommonMark](https://github.com/jgm/CommonMark), but others are possible. - Preferrably use a local copy for your live presentation.
# Agility! - This allows you to skip the explicit conversion step. - Just edit and reload as you make changes.
# Trick - The browser needs to think this is an HTML file, hence the `.html` extension. - The editor should think it's a Markdown file, hence the `-*-` tag.
